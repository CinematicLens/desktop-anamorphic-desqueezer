How to use (Professional guide)

This guide is written to match the app UI. Follow it exactly: import → ratio → (optional) CinemaScope → export.

1

Import media

Click Add Images / Add Videos, or drag & drop files into the app window. Your queue fills on the right.

2

Set the desqueeze factor

Select the correct ratio for your lens (ex: 1.33x, 1.5x, 1.8x, 2.0x). Use Custom for special adapters (ex: 1.60x).

3

Choose CinemaScope finishing (optional)

Enable CinemaScope export, choose a preset ratio like 2.39:1, and select Crop (fill) or Pad (fit).

4

Reframe with offsets (optional)

If your subject is too high/low after cropping, adjust Offset X/Y until framing is correct. Use the crop preview if available.

5

Set output folder

Confirm the Output Folder. Use Browse to route exports to your project deliverables directory.

6

Export with the right workflow

Export Selected = only checked items. Export All = entire queue. Use Merge + Export when you need one combined file.

Professional tips

• If you don’t know the lens factor, start with 1.33x for many modern anamorphic smartphone lenses, 1.5x for adapters, and 2.0x for classic anamorphic—then visually confirm circles look round.
• For CinemaScope delivery, 2.39:1 crop-to-fill is the most common finishing choice.
• Use Export Selected for QA: process 1–2 clips first, validate look, then run the full batch.

Workflow details

Use these options to match your lens workflow and deliver clean, share-ready results.

Export Selected (QC-first)

Tick ✓ for the items you want, run Export Selected, validate framing/ratio, then export the full batch.

CinemaScope crop vs pad

Crop fills the frame (most common). Pad preserves full image with letterboxing/pillarboxing.

Recommended delivery

For most cinema-style exports: desqueeze → 2.39:1 crop-to-fill → export MP4/PNG deliverables.
