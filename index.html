Complete feature set

Everything is designed for a fast and reliable desktop workflow: import, desqueeze, optional CinemaScope, and export.

Drag & drop import

Drop files anywhere in the window or use Add buttons. Works with MP4/MOV and JPG/PNG/TIF.

Desqueeze ratio presets

Select common anamorphic factors (via presets in the app). Clear readouts show exactly what’s applied.

Custom ratio input

When enabled in your build, you can type a custom ratio (e.g., 1.60) for unusual lenses and adapters.

CinemaScope export

Optional crop/pad after desqueeze for a clean CinemaScope-style output. Includes presets and fit modes.

Reframe offsets

Offset X/Y sliders help you reframe quickly before export — no extra editor needed.

Export workflows

Export Selected (checkbox-driven), Export All, and Merge + Export Selected for selected clips.

How to use (Quick Start)

A simple workflow that matches the app UI — import → choose ratio → optional CinemaScope → export.

1

Import files

Use Add Images / Add Videos or drag & drop files anywhere inside the app window.

2

Select export ratio

Pick your desqueeze ratio preset. If your build supports it, enter a custom ratio when needed.

3

Optional: enable CinemaScope

Turn on CinemaScope export, choose preset/fit mode, and adjust Offset X/Y for perfect framing.

4

Choose output folder

Confirm Output Folder or click Browse to change where results are saved.

5

Export Selected or Export All

Use checkboxes in the job list to export only what you want, or export everything in one run.

6

Monitor status

The right-side list shows Type/File/Status, and the bottom left displays export progress text.

Workflow details

Use these options to match your exact lens workflow and deliver clean, share-ready results.

Export Selected (checkbox-driven)

Tick ✓ for the items you want, then run Export Selected. Great for mixed import batches.

Per-clip CinemaScope (optional)

When enabled in your VM, per-clip export can preserve each clip’s crop/offset from Preview.

Merge + Export Selected

Select multiple clips and generate one merged output (depends on your VM implementation).
